# U9 — Projet Android Studio

## Ouvrir le projet
1. Installe Android Studio (https://developer.android.com/studio)
2. Ouvre Android Studio → "Open" → sélectionne ce dossier
3. Attends la synchronisation Gradle (1-2 min)
4. Connecte ton téléphone Android en USB ou lance un émulateur
5. Clique ▶ Run

## Générer un APK signé
Build → Generate Signed Bundle/APK → APK → crée une keystore → Release

## Prérequis
- Android Studio Giraffe ou plus récent
- SDK Android 24+ (Android 7.0)
- Java 17

## Mode hors-ligne
Le mode IA fonctionne sans internet.
Le mode multijoueur nécessite une connexion (Firebase).
